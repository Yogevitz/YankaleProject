####################################################################################
################################ Welcome to Boogle! ################################
####################################################################################

Boogle is a fast and efficient search engine.

Preconditions:

1. Python 2.7.15 64-bit version installed
2. PyStemmer 1.3 Package installed

In order to run Boogle:

1. Go to the Python27 directory in your computer (typically C:\Python27)
2. Open CMD in the directory (Shift+RightMouseClick -> Open PowerShell window here)
3. Type "python.exe " + path of "main.py" file from the directory of this readme
(should look something like this: python.exe C:\Users\yogev.s\PycharmProjects\Yankale\main.py)

####################################################################################

How to use Boogle:

1. Resources Path: Click the Browse button and choose a resources folder inwhich the corpus folder exists.
2. Save Path: Click the Browse button and choose a folder inwhich you wish Boogle to save the output files (indexes and postings).
3. Stemming Checkbox: Mark the checkbox to use Stemming, or leave it unmarked to use Boogle without Stemming.
4. Start Button: Click the Start button in order for Boogle to start working on your corpus.
5. Language List: After Boogle finishes working on your corpus, this list will display all the languages that are present in your corpus.
6. Reset Button: Click the Reset button to erase the produced indexes and posting files and start over.
7. Show Dictionary: Click the Show Dictionary button to open a window with the produced dictionary from your corpus.
8. Load Dictionary: Click the Load Dictionary button to load a dictionary of your own into Boogle.
9. Status: The label at the bottom shows you the current status of Boogle, please pay attention to the status label.

Good Luck!
